# SAP time & attendance automation

**Role:** Engineer / automation (project-style delivery)

## Summary

Automated reconciliation between **SAP time & attendance** logs and **production shift schedules** to reduce manual effort and errors.

## Scope

- **Python** scripts for **log reconciliation** against **shift plans**
- Automated calculation of **actual working hours**, **overtime**, and **attendance compliance**

## Value

- Repeatable, auditable **workforce** metrics aligned with **manufacturing or operations** rhythms
