# 计算

## 基本概念

* 矩阵：矩形的数组，即二维数组，其中向量和标量都是矩阵的特例
* 向量：是指1xn或者nx1的矩阵
* 标量：1x1的矩阵
* 数组：N维的数组，是矩阵的延伸
