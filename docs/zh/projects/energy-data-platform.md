# TB 级电力、企业与监管数据平台

**时间：** 2020.06 — 2023.09 · **角色：** 项目经理 · **团队：** 约 20 人开发团队 · **单位：** 南天（昆明）

## 概述

建设 **TB 级数据平台**，整合 **电厂**、**企业** 与 **监管** 侧数据，支撑 **能耗 / 能源** 分析、合规报送与跨机构数据视图（与简历表述一致）。

## 职责

- **项目与团队管理**（约 20 人）
- **系统架构** 与 **历史数据迁移**，保证互联系统间 **数据一致性与完整性**
- **数据工程：** TB 级 **集成与处理**、清洗与分析链路
- **过程改进：** 推动 **敏捷（Scrum）**，提升效率与客户满意度

## 技术背景

- 简历技能中的大数据组件：**Hadoop**、**SparkSQL**、**Hive**、**HBase**（按实际落地环境为准）
- 强调 **多源接入** 与面向分析与看板的 **治理与交付**

## 简历中的结果表述

- 异构源 **TB 级** 整合
- **数据清洗与分析** 流程优化
- 敏捷实践带来 **团队效率** 与 **客户满意度** 提升
