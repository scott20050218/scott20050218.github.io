# 零售数据、MCP 与门店分析

**背景：** 高校实习项目 · **角色：** 开发 / 分析

## 概述

围绕 **生产、销售、促销、门店** 等系统做 **数据整合** 与 **分析**，使用 **Cursor** 与 **自研 MCP** 统一数据访问、加快迭代。

## 交付物

- **MCP 服务**：整合生产、销售、促销、门店等后端数据
- **促销评估模型**：**ROI**、**lift**、**incrementality** 等指标
- **门店经营看板**：**SQL** + **Python**

## 技术栈（简历）

- **Python**、SQL、MCP 式集成
- 面向业务侧的 **分析与看板** 交付
