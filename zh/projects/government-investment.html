# 昆明政府投资平台

时间： 2023.10 — 2025.04 · 角色： 技术负责人（Lead Developer） · 单位： 南天（昆明）

# 概述

负责昆明市 政府投资类平台 的 架构、实现、测试与上线 全周期；通过 带团队、辅导初级开发 推动协作与工程规范。

# 技术范围

  • 架构： 基于 Spring Cloud 的可扩展 微服务
  • 端侧： Web、小程序、实时数据大屏 / 看板 等与同一套后端能力对接
  • 重点： 稳定发布、多渠道一体化、面向政务场景的可用性与运维要求

# 简历中的结果表述

  • 支撑 Web、小程序与 实时数据看板 的 并行交付
  • 技术牵头 + 人员培养
