# SAP 考勤与工时自动化

角色： 工程 / 自动化（项目制交付）

# 概述

将 SAP 考勤日志 与 生产排班 自动对账，减少人工核对与差错。

# 范围

  • Python 脚本：考勤记录 与 班次计划 对账
  • 自动计算 实际工时、加班 与 考勤合规 情况

# 价值

  • 可重复、可审计的 人力与工时 指标，贴合 制造或运营 排班节奏
