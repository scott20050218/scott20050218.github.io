# TB-scale power, enterprise & regulatory data platform

Period: 2020.06 — 2023.09 · Role: Project manager · Team: ~20 developers · Organization: Nantian, Kunming

# Summary

Directed a terabyte-scale data platform unifying power plant, enterprise, and regulatory data — relevant to energy / 能耗 analytics, compliance reporting, and cross-organizational visibility.

# Responsibilities

  • Program leadership for a 20-person development team
  • System architecture and legacy data migration with integrity across interconnected systems
  • Data engineering: ingestion and processing at TB scale, cleansing, and analysis pipelines
  • Process: Agile (Scrum) adoption to improve throughput and client satisfaction

# Technical context

  • Big-data style stack in CV: Hadoop, SparkSQL, Hive, HBase (use in your environment as applicable)
  • Emphasis on multi-source integration and governed handoff to analytics and dashboards

# Outcomes (from CV)

  • Integrated TB-scale heterogeneous sources
  • Streamlined data cleansing and analysis workflows
  • Stronger team efficiency and client satisfaction through Agile ways of working
